Encryption Locker Privacy Policy:

The application does not save passwords anywhere. You will lose access to encrypted data if you forget password used to encrypt it.

Windows 10 does not allow applications to access personal folders and files without permission. You will need to create a new folder or select an existing folder to give application permission to access folders and files inside it.

Application information page provides some Internet links about cryptography. Application does not use Internet to upload any information about use of application or any of your data.

Purpose of application is to protect privacy and data. Application does not have any backdoors or intentional vulnerabilities that allow access to encrypted data. Application uses Microsoft Advanced Encryption Standard (AES) methods. Password is only key to your encrypted data. Encrypted data is only as secure as password. Longer and complex passwords are much more resistant to hacking.



La aplicación no guarda contraseñas en ningún lado. Perderá el acceso a los datos cifrados si olvida la contraseña utilizada para encriptarlo.

Windows 10 no permite que las aplicaciones accedan a carpetas personales y archivos sin permiso. Tendrá que crear una nueva carpeta o seleccionar una carpeta existente para dar permiso a la aplicación para acceder a las carpetas y archivos dentro de ella.

La página de información de la aplicación proporciona algunos enlaces de Internet sobre criptografía. La aplicación no usa Internet para cargar información sobre el uso de la aplicación o cualquiera de sus datos.

El propósito de la aplicación es proteger la privacidad y los datos. La aplicación no tiene puertas traseras ni vulnerabilidades intencionales que permitan el acceso a datos encriptados. La aplicación utiliza los métodos estándar de cifrado avanzado de Microsoft (AES). La contraseña es solo la clave para sus datos encriptados. Los datos encriptados solo son tan seguros como la contraseña. Las contraseñas más largas y complejas son mucho más resistentes al pirateo.
