Encryption Locker Privacy Policy:

Application does not save passwords anywhere. You will lose access to encrypted data if you forget password used to encrypt it.

Windows 10 does not allow applications to access personal folders and files without your permission. Create new folder or select existing folder to give application permission to access items in that folder. < br>
Application information page provides some Internet links about cryptography. Application does not use Internet to upload any of your data or information about your use of application.

Purpose of application is to protect your data. Application does not have any backdoors or any intentional vulnerabilities that allow access to your encrypted data. Application uses Microsoft Advanced Encryption Standard (AES) methods. Your password is only way to access your encrypted data. Longer and complex passwords are much more resistant to hacking.



La aplicación no guarda contraseñas en ninguna parte. Perderá el acceso a los datos cifrados si olvida la contraseña utilizada para cifrarlos.

Windows 10 no permite que las aplicaciones accedan a carpetas y archivos personales sin su permiso. Cree una nueva carpeta o seleccione una carpeta existente para otorgar permiso a la aplicación para acceder a los elementos de esa carpeta.
