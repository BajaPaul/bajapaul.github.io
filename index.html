Privacy Policy for applications ‘Encryption Locker’ and ‘Archive & Extract’

More information about these applications can be found at:
Encryption Locker: https://www.microsoft.com/store/apps/9N5D1RB6H1JV
Archive & Extract: https://www.microsoft.com/store/apps/9NKF6BQXNDFW


English

Windows 10 does not allow applications to access personal folders and files without your explicit permission. These two applications require you to select a folder for application use. Folder selection provides applications read-write access to data in that folder and any other folders below it.

These applications require read-write access to data so they can create, rename, and delete folders and files as needed to complete their tasks. These applications do not transmit any of your data to third parties. Applications do not have backdoors or intentional vulnerabilities that allow others to access your data.

Encryption Locker application does not save your passwords anywhere. You will lose access to your encrypted data if you forget password used to encrypt it. This application uses Microsoft Advanced Encryption Standard (AES) methods.

简体中文

Windows 10 不允许应用程序在未经您明确许可的情况下访问个人文件夹和文件。 这两个应用程序要求您选择一个文件夹供应用程序使用。 文件夹选择提供应用程序对该文件夹中的数据及其下面的任何其他文件夹的读写访问权限。

这些应用程序需要读取写入数据访问权限,以便根据需要创建、重命名和删除文件夹和文件以完成任务。 这些应用程序不会将您的任何数据传输给第三方。 应用程序没有允许其他人访问您的数据的后门或故意漏洞。

应用程序加密锁不会将您的密码保存在任何位置。 如果您忘记了用于加密加密数据的密码,您将无法访问加密数据。 此应用程序使用微软高级加密标准 (AES) 方法。
