Privacy Policy for applications ‘Encryption Locker’ and ‘Archive & Extract’.
Both programmed by Paul Ghilino.

More information about these applications can be found at:
Encryption Locker: https://www.microsoft.com/store/apps/9N5D1RB6H1JV
Archive & Extract: https://www.microsoft.com/store/apps/9NKF6BQXNDFW


Windows 10 does not allow applications to access personal folders and files without your explicit permission. These two applications require you to select a folder for application use. Folder selection provides applications read-write access to data in that folder and any other folders below it.

These applications require read-write access to data so they can create, rename, and delete folders and files as needed to complete their tasks. These applications do not transmit any of your data to third parties. Applications do not have backdoors or intentional vulnerabilities that allow others to access your data.

Encryption Locker application does not save your passwords anywhere. You will lose access to your encrypted data if you forget password used to encrypt it. This application uses Microsoft Advanced Encryption Standard (AES) methods.


Windows 10 no permite que las aplicaciones accedan a carpetas y archivos personales sin su permiso explícito. Estas dos aplicaciones requieren que seleccione una carpeta para el uso de la aplicación. La selección de carpetas proporciona a las aplicaciones acceso de lectura y escritura a los datos de esa carpeta y a cualquier otra carpeta debajo de ella.

Estas aplicaciones requieren acceso de lectura y escritura a los datos para que puedan crear, renombrar y eliminar carpetas y archivos según sea necesario para completar sus tareas. Estas aplicaciones no transmiten ninguno de sus datos a terceros. Las aplicaciones no tienen puertas traseras o vulnerabilidades intencionales que permiten que otros accedan a sus datos.

La aplicación Locking Encryption no guarda tus contraseñas en ningún lugar. Perderá el acceso a sus datos cifrados si olvida la contraseña utilizada para cifrarlos. Esta aplicación utiliza los métodos de Microsoft Advanced Encryption Standard (AES).


विंडोज 10 आपके स्पष्ट अनुमति के बिना व्यक्तिगत फ़ोल्डर्स और फ़ाइलों तक पहुंचने की अनुमति नहीं देता है। इन दो अनुप्रयोगों के लिए आपको एप्लिकेशन उपयोग के लिए एक फ़ोल्डर का चयन करना होगा। फ़ोल्डर चयन एप्लिकेशन को उस फ़ोल्डर में डेटा और उसके नीचे किसी अन्य फ़ोल्डर में रीड-राइट एक्सेस प्रदान करता है।
